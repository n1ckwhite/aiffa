# Как писать полезные статьи в AIFFA

## Структура, которая работает

### 1) Начни с цели

Опиши, что читатель получит после прочтения, и для кого материал.

### 2) Дай контекст и проблему

Почему это важно? Где люди чаще всего ошибаются?

## Пример кода

```ts
type Article = {
  title: string;
  description: string;
  sections: string[];
};

const buildSummary = (article: Article) => {
  if (!article.title) return "Без названия";
  return `${article.title}: ${article.description}`;
};
```
